Question-1: What are the different ways to select an element in the DOM?

We can use "getElementById" for select an specific id of the HTML element
use "getElementsByClassName" for select the Collection of same type elements by the class name of the HTML elements
use "getElementsByTagName" for select the HTML Collection, here elements are specified by the tagname
use "querySelector" for selecting first matching element with any class or id
use "querySelectorAll" for selecting all matching elements by using any classname or html tag

Question-4: What is event bubbling in the DOM?

Event Bubbling is a mechanism where the event start from the target element, and it go through upper position, from target elements it hits the parent elements, after that it hits the parent's parent elements, following this way it hits to the last parent of the target elements
